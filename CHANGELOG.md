# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-07-07
### Added
- Feature **Remove Swift Injection** with toolbar icon (Trash icon) and keyboard shortcut (`Cmd + Alt + U` on Mac, `Ctrl + Alt + U` on Windows).

## [1.0.0] - 2026-07-07
### Added
- First release of **Swift SwiftUI Injection Generator**.
- Alphabetical `import Inject` insertion.
- Indentation-aware `@ObserveInjection var inject` insertion inside main struct.
- Braces-matched `.enableInjection()` modifier insertion at the end of the View's `body` property.
- Editor title action icon (Zap) and default keyboard shortcut (`Cmd + Alt + I` on Mac, `Ctrl + Alt + I` on Windows).
